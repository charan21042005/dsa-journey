# Operators in C++
Operators are **symbols** that tell the compiler to perform specific
operations on operands (values or variables).

Operators are one of the **most important building blocks** in C++ because:
- They help perform calculations
- They control logic and conditions
- They are heavily used in loops and DSA problems

## üîπ C++ Operator Types
C++ operators are classified based on:
1. **Purpose** (Arithmetic, Relational, Logical, etc.)
2. **Number of operands** they work on

In this section, we focus on **Arithmetic Operators**, which are the foundation
for calculations and logic building.

## üîπ 1. Arithmetic Operators in C++

Arithmetic operators are used to perform **basic mathematical operations**.

### üìò List of Arithmetic Operators

| Operator | Name | Description |
|-------|------|------------|
| `+` | Addition | Adds two operands |
| `-` | Subtraction | Subtracts right operand from left |
| `*` | Multiplication | Multiplies two operands |
| `/` | Division | Divides left operand by right |
| `%` | Modulo | Returns remainder |
| `++` | Increment | Increases value by 1 |
| `--` | Decrement | Decreases value by 1 |

### 1Ô∏è‚É£ Addition Operator (`+`)

```cpp
int a = 10;
int b = 5;
int sum = a + b;

cout << sum;
```
* Adds two operands
* Works with integers and floating-point numbers

### 2Ô∏è‚É£ Subtraction Operator (`-`)
```cpp
int a = 10;
int b = 4;
int diff = a - b;

cout << diff;
```
* Subtracts right operand from left

### 3Ô∏è‚É£ Multiplication Operator (`*`)
```cpp
int a = 6;
int b = 5;
int product = a * b;

cout << product;
```
* Multiplies two values

### 4Ô∏è‚É£ Division Operator (`/`)
```cpp
int a = 10;
int b = 3;

cout << a / b;
```
Output:
```cpp
3
```
Explanation:
* Integer division discards the decimal part

**Division with Floating-Point Values**
```cpp
float a = 10;
float b = 3;

cout << a / b;
```
Output:
```cpp
3.33333
```
* Division works with floating-point numbers

### 5Ô∏è‚É£ Modulo Operator (`%`)
The modulo operator returns the **remainder** after division.
```cpp
int a = 10;
int b = 3;

cout << a % b;
```
Output:
```cpp
1
```
**üî¥ Important Point (VERY IMPORTANT)**
> The modulo operator (`%`) works only with integers.

‚ùå Invalid:
```cpp
float a = 10.5;
float b = 2.5;
// cout << a % b;  ‚ùå Error
```
‚úî Valid:
```cpp
int a = 10;
int b = 3;
cout << a % b;
```

### 6Ô∏è‚É£ Increment Operator (`++`)
The increment operator increases a variable‚Äôs value by 1.

There are two types:
1. Pre-increment (++a)
2. Post-increment (a++)

### Pre-Increment (`++a`)
```cpp
int a = 5;
int b = ++a;

cout << a << " " << b;
```
Output:
```cpp
6 6
```
Explanation:
* Value of `a` is incremented first
* Then assigned to `b`

### Post-Increment (`a++`)
```cpp
int a = 5;
int b = a++;

cout << a << " " << b;
```
Output:
```cpp
6 5
```
Explanation:
* Value of `a` is assigned first
* Then incremented

### 7Ô∏è‚É£ Decrement Operator (--)
The decrement operator decreases a variable‚Äôs value by 1.

### Pre-Decrement (`--a`)
```cpp
int a = 5;
int b = --a;

cout << a << " " << b;
```
Output:
```cpp
4 4
```

### Post-Decrement (`a--`)
```cpp
int a = 5;
int b = a--;

cout << a << " " << b;
```
Output:
```cpp
4 5
```

## üîπ Operators Based on Number of Operands
Operators are also classified based on how many **operands** they work with.

### 1Ô∏è‚É£ Unary Operators
Unary operators work on **a single operand**.

Examples
```cpp
int a = 5;
cout << ++a;
```
Common unary operators:
* `++`
* `--`
* `!` (logical NOT)
* `-` (unary minus)

### 2Ô∏è‚É£ Binary Operators
Binary operators work on **two operands**.

Examples
```cpp
int a = 10;
int b = 5;

cout << a + b;
```
Most arithmetic, relational, and logical operators are binary.

### 3Ô∏è‚É£ Ternary Operator (`?:`)

The ternary operator works on **three operands**.

**Syntax**
```cpp
condition ? expression1 : expression2;
```
Example
```cpp
int a = 10;
int b = 20;

int max = (a > b) ? a : b;
cout << max;
```
Explanation:
* If condition is true ‚Üí `expression1`
* Else ‚Üí `expression2`

### üîπ Summary
* Operators perform operations on variables and values
* Arithmetic operators handle calculations
* `%` works only with integers
* `++a` and `a++` behave differently
* Operators are classified as:
  * Unary
  * Binary
  * Ternary

Understanding operators is **crucial for logic building**, loop control,
and solving **DSA problems efficiently**.

## üîπ 2. Relational Operators in C++
Relational operators are used to **compare two values or variables**.
They return a **boolean result**:
- `true` (1)
- `false` (0)

These operators are heavily used in:
- Conditions
- Loops
- Decision making

### üìò List of Relational Operators
| Operator | Name | Description |
|-------|------|------------|
| `==` | Equal to | Checks if two values are equal |
| `!=` | Not equal to | Checks if values are not equal |
| `>` | Greater than | Checks if left > right |
| `<` | Less than | Checks if left < right |
| `>=` | Greater than or equal to | Checks if left ‚â• right |
| `<=` | Less than or equal to | Checks if left ‚â§ right |

Example
```cpp
int a = 10;
int b = 20;

cout << (a == b);   // 0
cout << (a != b);   // 1
cout << (a > b);    // 0
cout << (a < b);    // 1
cout << (a >= b);   // 0
cout << (a <= b);   // 1
```
**Important Note**
‚ùå Do NOT confuse:
```cpp
a = b;   // Assignment
a == b;  // Comparison
```

## üîπ 3. Logical Operators in C++
Logical operators are used to **combine or modify conditions** and always return
a boolean result (`true` or `false`).

### üìò List of Logical Operators in C++
| Operator | Name | Description |
|---------|------|-------------|
| `&&` | Logical AND | Returns `true` if **both conditions** are true |
| `\|\|` | Logical OR | Returns `true` if **at least one condition** is true |
| `!` | Logical NOT | Reverses the result of a condition |

### Logical AND (`&&`)
```cpp
int age = 20;

if (age >= 18 && age <= 60) {
    cout << "Eligible";
}
```
‚úî Both conditions must be true

### Logical OR (`||`)
```cpp
int day = 6;

if (day == 6 || day == 7) {
    cout << "Weekend";
}
```
‚úî At least one condition must be true

### Logical NOT (`!`)
```cpp
bool isLoggedIn = false;

if (!isLoggedIn) {
    cout << "Please login";
}
```
‚úî Reverses the boolean value

## üîπ 4. Bitwise Operators in C++

Bitwise operators work at the **bit level (0s and 1s)**.
They are commonly used in:
* Low-level programming
* Optimization
* Competitive programming

### üìò Bitwise Operators List
| Operator | Name             | Description       |        
| -------- | ---------------- | ----------------- |
| `&`      | Bitwise AND      | AND operation     |    
| `        | `                | Bitwise OR        | 
| `^`      | Bitwise XOR      | Exclusive OR      |        
| `<<`     | Left Shift       | Shifts bits left  |             
| `>>`     | Right Shift      | Shifts bits right |              
| `~`      | One‚Äôs Complement | Inverts bits      |       

### Bitwise AND (`&`)
```cpp
int a = 5;   // 0101
int b = 3;   // 0011

cout << (a & b);   // 1
```

### Bitwise OR (`|`)
```cpp
cout << (a | b);   // 7
```

### Bitwise XOR (`^`)
```cpp
cout << (a ^ b);   // 6
```

### Left Shift (`<<`)
```cpp
int x = 5;
cout << (x << 1);  // 10
```
Explanation:
* Shifts bits left
* Multiplies by 2

### Right Shift (`>>`)
```cpp
cout << (x >> 1);  // 2
```
Explanation:
* Shifts bits right
* Divides by 2

### One‚Äôs Complement (`~`)
```cpp
int x = 5;
cout << ~x;
```
Explanation:
* Flips all bits
* Result is negative due to two‚Äôs complement

## üîπ Operator Associativity in C++

Associativity determines the direction of **evaluation**
when operators of the same precedence appear together.

![Operator Associativity Example](../Images/6.png)

### Types of Associativity
| Associativity | Direction                  |
| ------------- | -------------------------- |
| Left to Right | Most operators             |
| Right to Left | Assignment, unary, ternary |

### Left-to-Right Associativity Example
```cpp
int x = 10 - 5 - 2;
cout << x;
```
Evaluation:

```cpp
int a, b, c;
a = b = c = 10;
```
Evaluation:
```cpp
a = (b = (c = 10))
```

## üîπ Decision Making in C++

Decision making allows a program to **choose different paths of execution**

- Logic building

### üîπ `if` Conditional Statement

The `if` statement executes a block of code **only if the condition is true**.

![Flowchart for if Condition in C++](../Images/10.png)


### Syntax

```cpp

    // code
}

```

```cpp

    int age = 20;

    if (age >= 18) {
        cout << "You are eligible to vote";
    }

    return 0;
}
```
### Important Note

If there is **only one statement**, curly braces `{ }` can be skipped.
```cpp

    cout << "You are eligible to vote";
```
‚ö†Ô∏è However, using **braces is a good practice** to avoid logical errors.

### üîπ `switch` Statement

The `switch` statement is used when a variable is compared against
**multiple constant values**.

It is often cleaner than long `if-else` ladders.

![Flowchart for switch Statement in C++](../Images/14.png)

### Syntax
```cpp
switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
}
```
### Example (Full Program)
```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;

    cout << "Enter a number (1-3): ";
    cin >> choice;

    switch (choice) {
        case 1:
            cout << "You selected One";
            break;
        case 2:
            cout << "You selected Two";
            break;
        case 3:
            cout << "You selected Three";
            break;
        default:
            cout << "Invalid choice";
    }

    return 0;
}
```
### Important Notes about `switch`
* `break` prevents fall-through
* `case` values must be constants
* `default` executes if no case matches

## üîπ Jump Statements in C++

Jump statements are used to **alter the normal flow of execution** of a program.
Instead of executing statements sequentially, jump statements allow the program
to **jump to another part of the code**.

They are commonly used in:
- Loops
- Functions
- Conditional logic
- Controlling program execution

C++ provides the following jump statements:
1. `continue`
2. `break`
3. `return`
4. `goto`

---

### üîπ 1. `continue` Statement

The `continue` statement is used **inside loops**.
It skips the **current iteration** and moves directly to the **next iteration** of the loop.

![Flowchart for continue Statement in C++](../Images/15.png)

### How `continue` works
- The condition is checked
- If `continue` is encountered:
  - Remaining code in the loop is skipped
  - Loop proceeds to the next iteration

### Example (Full Program)

```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;   // skips when i == 3
        }
        cout << i << " ";
    }
    return 0;
}
```
Output
```cpp
1 2 4 5
```

### Key Points
* Used only in loops
* Does NOT terminate the loop
* Skips the current iteration only

---

### üîπ 2. `break` Statement

The `break` statement is used to **immediately terminate a loop or switch statement**.

Once `break` is executed, control moves **outside the loop or switch block**.

![Flowchart for break Statement in C++](../Images/16.png)

### Example with Loop (Full Program)
```cpp
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break;   // exits the loop completely
        }
        cout << i << " ";
    }
    return 0;
}
```
Output
```cpp
1 2
```

### Example with `switch` (Full Program)
```cpp
#include <iostream>
using namespace std;

int main() {
    int choice = 2;

    switch (choice) {
        case 1:
            cout << "Option 1";
            break;
        case 2:
            cout << "Option 2";
            break;
        default:
            cout << "Invalid choice";
    }

    return 0;
}
```
### Key Points
* Terminates the loop or switch
* Control exits the current block
* Commonly used in `switch` statements

### Difference Between `break` and `continue`
| Feature         | `break`         | `continue`              |
| --------------- | --------------- | ----------------------- |
| Effect          | Terminates loop | Skips current iteration |
| Loop continues? | ‚ùå No            | ‚úÖ Yes                   |
| Used in         | Loops & switch  | Loops only              |
